# PRISM Extension Assets

This directory contains icons and other assets for the PRISM Chrome extension.

## Required Icons
- icon16.png (16x16) - Toolbar icon
- icon48.png (48x48) - Extension management page
- icon128.png (128x128) - Chrome Web Store

## Placeholder Icons
Temporary placeholder icons are provided. Replace with professionally designed icons before release.

## Asset Guidelines
- Use consistent color scheme (blue/purple gradient)
- Include shield symbol for security/privacy theme
- Maintain high contrast for accessibility
- Follow Chrome extension icon guidelines
