## üé® Screenshot Tips

### Technical Requirements
- **Resolution**: 1280x800 (recommended) or 640x400
- **Format**: PNG or JPG
- **Max file size**: 5MB per screenshot
- **Orientation**: Landscape (horizontal)

### Quality Guidelines
‚úÖ **DO**:
- Use high-resolution displays for crisp screenshots
- Ensure text is readable
- Show actual PRISM interface (no mockups)
- Use real websites for testing
- Capture full popup window
- Show emerald green theme clearly
- Include version number where visible

‚ùå **DON'T**:
- Blur or pixelate important text
- Edit screenshots heavily (minor cropping OK)
- Show personal information
- Use development mode or console errors
- Include browser bookmarks or personal tabs
- Show outdated UI or version numbers

### Editing (Optional)
You can:
- Crop to focus on PRISM interface
- Add subtle border for clarity
- Adjust brightness/contrast slightly
- Add arrow pointing to key feature (sparingly)
